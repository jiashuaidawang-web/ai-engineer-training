# LlamaIndex 学习路线图（面向 Java 程序员）

## 学习路径

| 序号 | 课程 | 核心概念 | 类比 Java |
|------|------|----------|-----------|
| 01 | Node / 文本切片 | Document → Node，5种Splitter | String.split() 的智能版 |
| 02 | Settings / 全局配置 | LLM + Embedding 模型接入 | application.yml / Spring Bean 注入 |
| 03 | Reader / 文档加载 | 从文件读取 Document | Apache Tika / POI 读取文件 |
| 04 | Vector Store / 向量存储 | FAISS / Chroma 持久化向量 | MySQL / MongoDB 数据存储 |
| 05 | Index + Retriever / 索引与检索 | VectorStoreIndex 构建与搜索 | Repository.findByKeyword() |
| 06 | Postprocessor / 重排序 | 相似度过滤 + Reranker 精排 | 二级排序 / 相关性打分 |
| 07 | Query Engine / 查询引擎 | 检索 + 合成 = 生成答案 | Service 层：DTO → 响应 |
| 08 | Ingestion Pipeline / 摄取管线 | Reader → Splitter → Embed → Store 一键编排 | Spring Batch 作业 |
| 09 | Chat Engine + Memory / 对话引擎 | 多轮对话记忆 | HttpSession / 会话状态机 |
| 10 | Agent + Tools / 智能体 | LLM 自主调用工具 | 策略模式 + 工厂 |
| 11 | 综合项目：完整 RAG 系统 | 串联全部知识 | CRUD 全栈应用 |

## 每课风格

- 类比 Java 概念，帮助理解
- 每行代码都有详细注释
- 先讲"是什么"，再讲"怎么用"，最后讲"为什么"
- 提供可运行的完整代码
